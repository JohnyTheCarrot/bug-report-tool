Create a Bug Report
Alert:
  • Please note this tool is still using the old bot workflow
  • The workflow will be updated once the server revamp comes out

Before we start
  • Make sure you're reliably able to reproduce the issue. If you don't have reliable reproduction steps please contact support instead
  • Make sure to check the relevant Trello board to see if your bug hasn't been already reported.

  • Desktop: https://trello.com/b/AExxR9lU/canary-bugs
  • Android: https://trello.com/b/Vqrkz3KO/android-beta-bugs
  • IOS: https://trello.com/b/vLPlnX60/ios-testflight-bugs
  • Store: https://trello.com/b/Fb0LyRnD/store-bugs
  • Website: https://trello.com/b/3ubHh1rm/website-bug-board
  • Overlay: https://trello.com/b/k6jr4NyY/overlay-bugs

  • If you didn't find it to be reported on Trello you can also use the relevent report channel and use the search command inside of Discord.
  • Please note that this site will NOT submit the bug for you.
  • If you want to attach an image/video to your report please ask a Bug Hunter or a Helper to attach it for you

description In a few sentences describe what is your issue. Avoid using word as 'glitch' or 'bug'
create Try to make your reproduction steps as easy to follow as possible.
